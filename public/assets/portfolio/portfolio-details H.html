Je vous présente les détails de mon Projet

  • Projet réaliser en langage Golang.
  • Ce projet est basé sur le célèbre HangMan Game Vous devrez créer un référentiel privé avec le nom hangman-classic.
  • Créez un programme pendu qui prendra un fichier en paramètre. Créez un fichier words.txt qui contient un tas de mots avec lesquels le programme jouera.
  • Vous aurez 10 tentatives pour terminer le jeu. Tout d'abord, le programme choisira au hasard un mot dans le fichier.
  • Le programme révélera n lettres aléatoires dans le mot, où n est le len (mot) / 2 - 1. Le programme lira l'entrée standard pour suggérer une lettre.
  • Si la lettre n'est pas présente, il imprimera un message d'erreur et le nombre de tentatives diminuera (10->9->…0).
  • Si la lettre est présente, il révélera toutes les lettres correspondantes dans le mot. Le programme continue jusqu'à ce que le mot soit trouvé ou que le nombre de tentatives soit égal à 0.
